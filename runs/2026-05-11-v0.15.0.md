## Features shipped

### #18 Four-tier project memory (Gemini CLI pattern)

`load_project_layers(cwd)` returns ordered `(path, content)` pairs from
four tiers:

1. `~/.open-code/OPEN_CODE.md` (global personal defaults)
2. Each ancestor's `OPEN_CODE.md`, top-down (monorepo roots)
3. `<cwd>/OPEN_CODE.md` (project, committed)
4. `<cwd>/.open-code/MEMORY.md` (private/uncommitted)

`build_system_instruction_layered(layers)` concatenates them under
labeled `## Project context from <path>` section headers. Missing
files silently skipped. Each layer capped at 30KB; total cap raised
50KB → 100KB.

Back-compat: `load_project_context(cwd)` + `build_system_instruction`
preserved as shims so external callers don't break.

### #15 Effort levels

`--effort low|medium|high|xhigh` flag + `effort:` settings field +
`/effort` REPL command. Maps to a `thinking_budget` int:

| Effort | Budget |
|--------|--------|
| low | 0 (no extended reasoning) |
| medium (default) | 512 |
| high | 4096 |
| xhigh | 16384 |

Passed via `types.ThinkingConfig` if the SDK supports it (guarded by
`getattr`). Models that don't honor thinking budgets ignore it
silently — no breakage.

### #16 Extended thinking / `ultrathink`

In-prompt marker: scan the task text for the literal token
`ultrathink` (case-insensitive, word-boundary'd). When present:
- Strip it from the prompt the model sees
- Bump THIS turn's `thinking_budget` to `ULTRATHINK_BUDGET = 32768`
- Don't affect `settings.effort` — one-off escape hatch

Word-boundary regex `\bultrathink\b` so `ultrathinker` doesn't trigger.

### #14 Status line (bonus, threaded through)

`--statusline` flag enables a per-iter footer to stderr:
```
  [model=gemini-3.1-flash-lite-preview effort=high iter=2 in_tok=2973 out_tok=147 tool_errs=0]
```

Off by default. Composes with `--show-metrics` (cumulative summary)
and the per-iter `[iter N] calling …` trace.

## Honest trade-offs

- **`ThinkingConfig` is best-effort**. If the user's SDK is older or
  the model doesn't reason, the value is ignored silently. No way
  for v0.15 to verify the budget was honored from outside the model.
- **Status line goes to stderr, not a re-drawing TUI**. Each iter
  appends a new line. A "real" status line would use ANSI cursor
  control to update a single line at the bottom. Deferred.
- **Ultrathink is detected anywhere in the prompt**, not just at the
  start. If a user writes "explain the ultrathink algorithm", we
  strip the word and bump the budget. Trade-off: simpler regex vs.
  rare false positives. Documented in code.
- **Four-tier memory total cap is 100KB** — could be an issue if all
  four layers are at their 30KB cap. The 100KB total still leaves
  room for the system instruction itself.
